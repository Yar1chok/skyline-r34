# Skyline R34

Интерактивная 3D-презентация Nissan Skyline GT-R R34 в браузере: модель автомобиля стоит на месте, а виртуальная камера облетает её по мере прокрутки страницы, попутно показывая характеристики — двигатель, тормозную систему, салон.

## Возможности

- **Скролл-камера.** 9 ключевых кадров ведут камеру от вида сбоку через переднее колесо и салон к виду сзади; на каждом отрезке показываются соответствующие технические характеристики.
- **Тёмная / светлая тема.** Ночь со звёздами и луной или закат с солнцем — меняются свет, туман, цвета травы и земли.
- **Русский / английский язык.** Переключатель в шапке меняет все тексты интерфейса, включая характеристики машины.
- **Панель настроек сцены.** Живая регулировка тумана, освещения, цветов травы и земли, теней и авто-вращения камеры.
- **Адаптивное качество рендера.** Сцена сама снижает разрешение, если FPS проседает на слабом железе.

## Стек

- [Angular 21](https://angular.dev) — standalone-компоненты, signals, без Zone.js
- [Three.js](https://threejs.org) — 3D-сцена, GLTF-модели, кастомный GLSL-шейдер травы
- [Vitest](https://vitest.dev) — юнит-тесты
- TypeScript (strict), SCSS, Prettier

## Быстрый старт

```bash
npm install
npm start        # http://localhost:4200
```

Другие команды:

```bash
npm run build    # прод-сборка в dist/
npm run watch    # сборка с автопересборкой (dev-режим)
npm test         # юнит-тесты (Vitest)
```

## Структура проекта

```
src/app/
├── pages/home/              # единственная страница приложения
└── shared/
    ├── components/          # лоадер, панель настроек сцены
    ├── services/             # оркестратор сцены + camera/lighting/environment/car-model, тема, язык
    ├── const/                # константы сцены, секции, переводы
    ├── types/                # типы (кейфреймы, секции, переводы)
    ├── utils/                # шейдеры травы, билдеры сцены, математика
    └── styles/               # переменные темы и общий ui-kit
```

3D-логика разбита на отдельные сервисы по ответственности: `SceneCameraService` — движение камеры, `SceneLightingService` — весь свет, `SceneEnvironmentService` — окружение (дорога, трава, деревья, туман), `CarModelService` — загрузка и оптимизация GLB-модели. `CarSceneService` их координирует.
